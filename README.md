QGM: Toolbox for Quantum Gas Microscope experiments
================================

Required Packages
------------

* Python >=3.6
* numpy  >=1.18
* scipy  >= 1.3.2
* sympy  >=1.5.1
* pandas  >=0.25.3
* matplotlib  >= 3.1.1
* tqdm >= 4.42.0

Installation
------------

```bash
python setup.py install
```
