QGM: Toolbox for Quantum Gas Microscope experiments
================================

Required Packages
------------

* Python >=3.6
* numpy  >=1.18
* scipy  >= 1.3.2
* sympy  >=1.5.1
* pandas  >=0.25.3
* matplotlib  >= 3.1.1
* tqdm >= 4.42.0
* [astropy](https://www.astropy.org/) >= 4.0.1
* [photutils](https://photutils.readthedocs.io/en/stable/) >= 0.7.2

Installation
------------

```bash
python setup.py install
```
